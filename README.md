# YpackFood
Мобильное приложение для заказа еды в ресторане "Боярин"
Комплексный проект, выполнялся совместно с бэкенд-разработчиком.

Используемые технологии:
  - Kotlin
  - Jetpack Compose
  - MVVM
  - Паттерны проектирования
  - LiveData
  - Coroutines
  - Room
  - Datastore
  - SQLite
  - Retrofit
  - Moshi, GSON
  - JUnit
